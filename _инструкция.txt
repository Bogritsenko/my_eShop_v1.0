ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ
=================
- Польностью готовый интернет магазин условных товаров
  сделан на архитектуре MVC
  и поэтому легко расширяемый.
- Реализована технология  drag and drop 
  для добавления товаров в корзину,
  легко добавить свои категории, в данный момент товар 
  разбит на 2 категории.
- используеться php5, javascript, jquery, mysql, html5
- одна точка входа - index.php
- три основные папки: model, view, controller
- модель, по менеджеру на каждую группу задач.
- контроллеры, по штуке на каждую страницу.
- папка script - для клиентских сценариев на javascript и jquery
- в каждой папке документация для каждого класса сделана с помощью классов отражения

УСТАНОВКА
=========
1. Откройте файл Model_Driver.php и установите ваши настройки для подключения к базе данных.
2. Выполните скрипт импортирования создания объектов БД e-shop.sql .
3. Сайт готов к работе.
